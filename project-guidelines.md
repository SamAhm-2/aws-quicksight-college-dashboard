# Project Guidelines

## Assignment Overview

**Project Title:** From College Boards to Dashboards: Providing Data Insights to a Community College

**Objective:** Use AWS QuickSight to analyze student enrollment data and deliver actionable recommendations to improve professor evaluations while controlling course costs.

## Business Context

Regional Community College's Board of Directors sought to understand:
- Which professors earn the best evaluations
- What factors influence evaluation scores
- How to improve student satisfaction without raising costs

### Dataset Preparation
- Configure refresh schedules
- Create calculated fields (Student Type classification)
- Transform and rename data fields

### Analysis & Visualization
- Build interactive visuals using Amazon Q natural language queries
- Create charts analyzing enrollment, evaluations, and costs
- Develop dashboard visualizations

### Topic Configuration
- Configure Amazon Q topic with 20+ data fields
- Create 3 named entities (Student Details, Course Details, Professor Evaluation)
- Develop verified Q&A pairs for business users

### Dashboard Development
- Publish interactive dashboard with Q&A functionality
- Enable data exploration for stakeholders

### Scenario Analysis
- Develop multi-step reasoning process
- Explore factors influencing professor evaluation scores
- Formulate data-driven recommendations

### Data Storytelling
- Create executive-level presentation
- Combine visuals with narrative
- Present actionable insights to Board of Directors

## Key Deliverables

1. Configured dataset with custom fields
2. Interactive analysis with visualizations
3. Amazon Q topic with verified questions
4. Published dashboard with Q&A capability
5. Scenario analysis with iterative questioning
6. Executive data story presentation

## Tools Used

- **AWS QuickSight** - Dashboard creation and visualization
- **Amazon Q** - Natural language data querying
- **Sample Dataset** - Student enrollment statistics

## Project Rubric Sections

1. Dataset Preparation and Transformation
2. Creating Visuals Using Amazon Q
3. Building Interactive Dashboards and Topics
4. Scenario Creation and Complex Questioning
5. Data Storytelling
